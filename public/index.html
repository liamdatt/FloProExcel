Why in Excel

Sees what you see

Excel is both your calculator and your interface. This gives your agent that same interface.

FloPro reads the actual workbook: the structure, the formatting, the formulas, your current selection. It doesn't need an export and you can see it work in real time. It's working in the same place you are.

Built for real work

Multiple session tabs, conversation history, auto-context on every turn, and a curated managed model list. Built-in tools help you cross-check assumptions quickly.

Between saves, every edit gets an automatic checkpoint with one-click undo.

Installation
Three steps, no dev tools

Download one file, drop it in a folder, open Excel.

1 Download the manifest

This small file tells Excel where to find FloPro. It's the only thing you need to download.

manifest.prod.xml

2 Copy it to Excel's add-in folder

Open Terminal and paste this. It creates the folder if needed and copies the file.

$ mkdir -p ~/Library/Containers/com.microsoft.Excel/Data/Documents/wef && \
  cp ~/Downloads/manifest.prod.xml ~/Library/Containers/com.microsoft.Excel/Data/Documents/wef/
Saved it somewhere other than Downloads? Change ~/Downloads/ to wherever you put it.

3 Open FloPro in Excel

  1. Quit Excel completely (⌘Q) and reopen it
  2. Go to Insert → My Add-ins
  3. Click FloPro Financial Modelling to open the sidebar
Heads up — FloPro was built and tested on macOS. It should work on Windows (the underlying tech is cross-platform) but hasn't been extensively tested. Worth a try! If you hit issues, open a GitHub issue.

Managed access
Start instantly with managed OpenRouter

FloPro is preconfigured. Users do not need provider logins, API keys, or proxy setup.

How it works Recommended

A server-side OpenRouter key is injected by the backend for every request. The model selector is restricted to FloPro's curated list for predictable performance and cost control.

For operators: set OPENROUTER_API_KEY on the VPS, deploy frontend + backend on the same origin, and users can start immediately. The code is open source.

Built-in Jamaica market MCP Included

FloPro preloads a managed MCP connection for Jamaican market data and enables it by default. Users can toggle it in Connections without editing URLs or tokens.
